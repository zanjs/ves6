### ves6

[![Greenkeeper badge](https://badges.greenkeeper.io/zanjs/ves6.svg)](https://greenkeeper.io/)

a cmd tool to generate `jsconfig.json` for javascript project

一个cmd工具来生成 vs code 的配置文件 支持 es6 语法检查

### install 
```bash
$ npm install ves6 -g
```
安装到全局 

### usage
```bash
$ ves6
```
在您的项目跟目录下执行 `ves6`

一切为了快捷
